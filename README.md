# Scientific Data Harvester

Backend сервис для сбора научных публикаций через OpenAlex API, хранения в PostgreSQL и предоставления REST API.

## Технологии
- Node.js
- Express
- PostgreSQL
- Prisma ORM
- Docker
- Swagger
- Jest (тестирование)

---

## Запуск проекта

```bash
docker compose up --build
